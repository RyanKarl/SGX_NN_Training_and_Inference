# LE source code   
The LE is part of Intel(R) Software Guard Extensions for Linux\* OS which is published in [linux-sgx](https://github.com/intel/linux-sgx/) Github repository. The libsgx_le.signed.so in prebuilt package is built by [le](https://github.com/intel/linux-sgx/tree/master/psw/ae/le) with tag [SGX 2.7.1](https://github.com/intel/linux-sgx/releases/tag/sgx_2.7.1) and signed by Intel.
